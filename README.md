## :wrench: Install and run

Open terminal:

```sh
# Clone this repo
git clone https://github.com/osvaldokalvaitir/reactjs-accessibility

# Entry in folder
cd reactjs-accessibility

# Install deps with pnpm
pnpm install

# Launch the app with pnpm
pnpm dev
```
